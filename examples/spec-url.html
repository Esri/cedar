Example - Specification from URL

Specification from URL

This chart uses a specification that is not one of the standard type (bar, scatter, bubble, pie, etc.) by passing in the URL to a specification JSON file via the `specification` parameter.

  // NOTE: this is not one of the standard chart types
  // so we pass a url to the specification JSON
  var chart = new Cedar({"specification":"../../../cedar/data/templates/time-dots.json"});

  var dataset = {
    "url":"https://maps2.dcgis.dc.gov/dcgis/rest/services/FEEDS/CDW_Feeds/MapServer/8",
    "mappings":{
      "time":   {"field":"REPORTDATETIME"},
      "aspect": {"field":"OFFENSE"},
      "facet":  {"field":"WARD"},
      "color":  {"field":"SHIFT"}
    }
  };
  chart.dataset = dataset;

  chart.show({
    "elementId": "#chart",
    height: 1000
  });
