Definition Json

A definition object is the combintion of a dataset and a specification into a single object. This streamlines the process of hydrating a chart, as the complete definition can be stored in a file, database, or other location, and the url passed into the cedar constructor.

This is also the internal structure used by Cedar.

Specification Format

{
  "specifiction": {...specification object...},
  "dataset":{ ... dataset object... }
}
