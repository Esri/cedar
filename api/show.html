.show( opt )

Renders a cedar chart object into the specified element in the DOM.

chart.show({elementId:"#chartDiv"})

Arguments

Name Type Required Description
elementId string yes Id of the DOM node to render the chart into.

Example Code

<div id='chartDiv'></div>

<script>
var chart = new Cedar({'specification': 'http://example.org/url_to_specification.json'})

//create and render a chart
chart.show({
  'elementId': '#chartDiv'
});

</script>
